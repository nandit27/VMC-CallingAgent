# Municipal Call Center Backend

AI-based multilingual call center system for municipal corporation complaint registration.

## Features
- Multi-language support (Gujarati, Hindi, English)
- Speech-to-text transcription
- AI-powered complaint classification
- Automated complaint registration
- Text-to-speech responses

## Tech Stack
- Node.js + Express
- Twilio Voice API
- OpenAI Whisper (STT)
- OpenAI GPT (LLM)
- OpenAI TTS

## Setup
1. Copy `.env.example` to `.env`
2. Fill in your API keys
3. Run `npm install`
4. Run `npm run dev`
